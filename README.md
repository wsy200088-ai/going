🎀 自然輸入法 注音CSV 產生器
一個跑在瀏覽器裡的小工具，讓你貼上一批中文詞彙、自動產生注音碼，並輸出可直接匯入「自然輸入法」個人詞庫的 CSV 備份檔。
> 解決自然輸入法批次加詞的痛點：備份格式檔不會自動配注音，每筆都必須手動標注音才能匯入成功。這個工具自動處理掉這件事。

🚀 使用方式
打開頁面，左邊文字框一行貼一個詞
```
   感冒
   流感
   偏頭痛
   阿茲海默症
   ```
按【產生注音】
右邊表格顯示結果，黃底列代表含破音字、紅底列代表轉換失敗
注音不對的，直接在表格欄位裡修改
按【下載 CSV 檔】或【複製內容】
打開自然輸入法的「詞語管理」（快速鍵 `Ctrl+9`）→ 點【匯入 CSV】→ 選剛下載的檔案 → 完成 ✅

聲調	符號	範例
一聲	（無）	`ㄍㄢ`
二聲	`ˊ`	`ㄍㄢˊ`
三聲	`ˇ`	`ㄍㄢˇ`
四聲	`ˋ`	`ㄍㄢˋ`

音節之間以連字號 `-` 連接，例如「感冒」= `ㄍㄢˇ-ㄇㄠˋ`，符合自然輸入法的 `鍵盤碼` 欄位格式。

⚠️ 注意事項
破音字一定要手動檢查：工具預設選最常用讀音，但個人打字習慣可能不同。例如「血」可以是 `ㄒㄧㄝˇ`（口語）或 `ㄒㄩㄝˋ`（書面）、「中」可以是 `ㄓㄨㄥ`（中心）或 `ㄓㄨㄥˋ`（中風）。
冷僻字可能失敗：若 pinyin-pro 字典未收錄，會顯示「注音對照失敗」，需手動填注音。
首次載入需要網路：開啟頁面時會從 cdnjs 載入 pinyin-pro（約 1 MB），載入後即使離線也能繼續使用。
此工具只處理「備份格式 CSV」，輸出檔案會被自然輸入法的「匯入 CSV」功能讀取，不適用於詞庫匯出檔的編輯。
匯入後重複的詞會被忽略，自然輸入法會自動判斷，原本就有的詞不會被覆蓋（除非選「全部覆蓋」）。

📂 自然輸入法 CSV 備份檔格式
工具產出的 CSV 結構如下：
```
<!--Going12ProfileBackUpCsv-Begin-->,
詞彙1,注音1
詞彙2,注音2
...
```
編碼：UTF-8 with BOM
換行：CRLF（\r\n）
第一列：固定為 `<!--Going12ProfileBackUpCsv-Begin-->,`
後續每列：`詞彙,連字號連接的注音`
注音欄不可留空（這正是此工具誕生的原因）

📝 開發背景
此工具誕生於批次匯入 113 筆人名到自然輸入法詞庫時遇到的痛點：
官方教學說「匯入時注音欄可留空、系統會自動配音」，但實測在備份格式的匯入路徑中注音欄不可留空，必須手動填入連字號連接的注音碼。經過多次測試後確認此規則，遂開發此工具自動化此流程。
> 注音碼／鍵盤碼：自然輸入法用來記錄「打什麼注音會跳出這個詞」的對應碼，例如打 `ㄍㄢˇ-ㄇㄠˋ` 跳出「感冒」。
