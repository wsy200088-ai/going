# 🎀 注音CSV 產生器

一個跑在瀏覽器裡的小工具，貼上中文詞彙、自動產生注音碼，**同步輸出三種輸入法的詞庫匯入檔**：

- **自然輸入法** 個人詞庫 CSV（備份格式）
- **華碩智慧輸入法** 使用者字典 TXT
- **微軟新注音** 使用者造詞 TXT

> 解決台灣三大注音輸入法批次加詞的痛點：三者格式規則完全不同（編碼、分隔符號、聲調表示、音節結構都不一樣），手動編輯曠日費時，這個工具一次搞定。

## 🚀 使用方式

1. 打開頁面，左邊文字框**一行貼一個詞**

   ```
   台灣
   台北市
   今天天氣真好
   ```

2. 按【**產生注音**】

3. 右邊表格同時顯示三種輸入法的注音，**黃底列代表含破音字、紅底列代表轉換失敗**

4. 注音不對的，直接在表格欄位裡修改（三欄可分別改）

5. 按對應的下載按鈕：自然 CSV ／ 華碩 TXT ／ 微軟 TXT，可全部下載也可只下載需要的

6. 打開對應輸入法的詞庫管理工具 → 匯入剛下載的檔案 → 完成 ✅

### 各輸入法的匯入路徑

| 輸入法 | 開啟詞庫管理 | 匯入功能 |
|---|---|---|
| 自然輸入法 | 快速鍵 `Ctrl + 9` | 詞語管理 → 匯入 CSV |
| 華碩智慧輸入法 | 設定 → 使用者字典 | 匯入字典檔 |
| 微軟新注音 | 控制台 → 語言 → 內容 → 使用者造詞 | 匯入造詞檔 |

## 📂 三種輸入法的格式對照

| 規則 | 自然輸入法 CSV | 華碩智慧 TXT | 微軟新注音 TXT |
|---|---|---|---|
| 副檔名 | `.csv` | `.txt` | `.txt` |
| 編碼 | UTF-8 **有 BOM** | UTF-8 **無 BOM** | **UTF-16 LE 有 BOM** |
| 換行 | CRLF | CRLF | CRLF |
| 詞與注音分隔 | `,` 逗號 | 2 個半形空格 | 2 個半形空格 |
| 注音音節之間 | `-` 連字號 | 直接相連 | 1 個半形空格 |
| 音節內結構 | 自由長度 | 自由長度 | **固定 4 字位置**（聲母／介音／韻母／聲調），缺位用全形空格 `　` 佔位 |
| 一聲標記 | 不標 | 標 `ˉ` | 標 `ˉ` |
| 開頭標記 | `<!--Going12ProfileBackUpCsv-Begin-->,` | 無 | 無 |
| 行尾 | 無 | 無 | **2 個半形空格** |

### 同一個詞，三種格式長這樣

```
詞彙：王詩雅

自然輸入法 CSV：
<!--Going12ProfileBackUpCsv-Begin-->,
王詩雅,ㄨㄤˊ-ㄕ-ㄧㄚˇ

華碩智慧輸入法 TXT：
王詩雅  ㄨㄤˊㄕˉㄧㄚˇ

微軟新注音 TXT：
王詩雅  　ㄨㄤˊ ㄕ　　ˉ 　ㄧㄚˇ  
```

（微軟格式中的 `　` 是全形空格，用來填補音節中沒有的聲母／介音／韻母位置）

## 🛠 技術說明

| 項目 | 內容 |
|---|---|
| 漢字 → 拼音 | [pinyin-pro](https://github.com/zh-lx/pinyin-pro) v3.27.0（透過 cdnjs 載入） |
| 拼音 → 注音 | 內建 410 筆音節對照表（透過 pypinyin 反查產生） |
| 三種格式輸出 | 自訂 `TONE_SUFFIX_NATURAL` / `TONE_SUFFIX_ASUS` / `TONE_SUFFIX_MSIME` 聲調表 |
| 微軟格式 4 位拆解 | 自訂 `bopomofoTo4Slot()` 把連續注音拆成聲母／介音／韻母／聲調四欄 |
| UTF-16 LE 編碼 | 自訂 `strToUtf16LE()` 把字串轉成 UTF-16 little-endian 位元組陣列 + BOM |
| 框架 | 純 HTML + CSS + JavaScript，無前端框架 |
| 字體基準 | `html { font-size: 18px }`（比瀏覽器預設 16px 稍大、好讀不破版） |
| 單位制 | 全頁採 rem，整體可按比例縮放 |
| 樣式 | 自訂 CSS，馬卡龍粉彩色系（白底淺色） |
| 部署 | 單一 `index.html`，可直接放靜態空間 / GitHub Pages |

### 注音聲調對應

| 聲調 | 自然 | 華碩 | 微軟 |
|---|---|---|---|
| 一聲 | 不標 | `ˉ` | `ˉ` |
| 二聲 | `ˊ` | `ˊ` | `ˊ` |
| 三聲 | `ˇ` | `ˇ` | `ˇ` |
| 四聲 | `ˋ` | `ˋ` | `ˋ` |
| 輕聲 | `˙` | `˙` | `˙` |

### 微軟新注音的音節 4 位拆解規則

每個注音音節固定 4 字位置：

| 位置 | 內容 | 範例（小 ㄒㄧㄠˇ） | 範例（沼 ㄓㄠˇ） |
|---|---|---|---|
| 1 | 聲母（21 個之一）或全形空格 | `ㄒ` | `ㄓ` |
| 2 | 介音（ㄧㄨㄩ 之一）或全形空格 | `ㄧ` | `　` |
| 3 | 韻母（13 個之一）或全形空格 | `ㄠ` | `ㄠ` |
| 4 | 聲調符號 | `ˇ` | `ˇ` |

## ⚠️ 注意事項

| 風險 | 說明與建議 |
|---|---|
| **破音字一定要手動檢查** | 工具預設選最常用讀音，但個人打字習慣可能不同。例如「血」可以是 `ㄒㄧㄝˇ`（口語）或 `ㄒㄩㄝˋ`（書面）、「中」可以是 `ㄓㄨㄥ`（中心）或 `ㄓㄨㄥˋ`（中風）、姓氏「許」應為 `ㄒㄩˇ`（pinyin-pro 預設為一聲，需手動改） |
| **冷僻字可能失敗** | 若 pinyin-pro 字典未收錄，會顯示「注音對照失敗」，需手動填注音 |
| **三種輸入法的破音字可能要分別校正** | 同一個破音字在三種輸入法的詞庫可能會選不同讀音，記得**三欄都檢查** |
| **首次載入需要網路** | 開啟頁面時會從 cdnjs 載入 pinyin-pro（約 1 MB），載入後即使離線也能繼續使用 |
| **本工具只處理備份／字典格式檔** | 各輸入法的「匯入詞庫」功能使用，不適用其他編輯場景 |
| **匯入後重複的詞會被忽略** | 三種輸入法都會自動判斷重複，原本就有的詞不會被覆蓋（除非選「全部覆蓋」） |
| **微軟新注音匯入後請以系統管理員身份操作** | Windows 對使用者造詞檔可能有權限限制，若匯入失敗可改用系統管理員執行匯入工具 |

## 📝 開發背景

此工具誕生於批次匯入 113 筆人名到自然輸入法詞庫時遇到的痛點：官方教學說「匯入時注音欄可留空、系統會自動配音」，但實測在備份格式的匯入路徑中**注音欄不可留空，必須手動填入連字號連接的注音碼**。

後續為了同時支援華碩智慧輸入法（格式規則完全不同）以及微軟新注音（UTF-16 編碼、4 位音節結構），擴充為三輸入法支援。

> 注音碼／鍵盤碼：輸入法用來記錄「打什麼注音會跳出這個詞」的對應碼，例如打 `ㄍㄢˇ-ㄇㄠˋ` 跳出「感冒」。
