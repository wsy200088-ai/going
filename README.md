# 🎀 注音CSV 產生器

一個跑在瀏覽器裡的小工具，貼上中文詞彙、自動產生注音碼，**同步輸出兩種輸入法的詞庫匯入檔**：

- **自然輸入法** 個人詞庫 CSV（備份格式）
- **華碩智慧輸入法** 使用者字典 TXT

> 解決台灣兩大注音輸入法批次加詞的痛點：兩者注音格式規則完全不同，手動編輯曠日費時，這個工具一次搞定。

## ✨ 功能特色

| 功能 | 說明 |
|---|---|
| **同步雙格式** | 一次貼詞，自動產出自然 CSV 與華碩 TXT 兩種格式 |
| **獨立編輯** | 兩種輸入法的注音欄各自可改，互不影響 |
| **破音字偵測** | 含多音字的詞自動標 ⚠ 黃底，提醒檢查 |
| **配色辨識** | 自然輸入法 = 綠色家族、華碩智慧輸入法 = 粉色家族，badge 與下載按鈕同色系一目了然 |
| **即時統計** | 總數 ／ 正常 ／ 含破音字 ／ 失敗 一目了然 |
| **標準格式輸出** | 兩種格式都嚴格遵守原輸入法備份檔的規範（編碼、BOM、換行、分隔符號） |
| **自動命名下載** | 自然 = `Going12Backup_Profile_YYYYMMDD_HHMM.csv`、華碩 = `UserDictionary_YYYYMMDD_HHMM.txt` |
| **重複自動去除** | 同樣的詞貼兩次只輸出一次 |
| **隱私安全** | 純前端運作，詞彙資料**不傳送到任何伺服器** |

## 🚀 使用方式

1. 打開頁面，左邊文字框**一行貼一個詞**

   ```
   台灣
   台北市
   今天天氣真好
   ```

2. 按【**產生注音**】

3. 右邊表格同時顯示兩種輸入法的注音，**黃底列代表含破音字、紅底列代表轉換失敗**

4. 注音不對的，直接在表格欄位裡修改（兩欄可分別改）

5. 按【**下載 自然輸入法 CSV**】或【**下載 華碩智慧 TXT**】，或兩個都下載

6. 打開對應輸入法的詞庫管理工具 → 匯入剛下載的檔案 → 完成 ✅

### 對應各輸入法的匯入路徑

| 輸入法 | 開啟詞庫管理 | 匯入功能 |
|---|---|---|
| 自然輸入法 | 快速鍵 `Ctrl + 9` | 詞語管理 → 匯入 CSV |
| 華碩智慧輸入法 | 設定 → 使用者字典 | 匯入字典檔 |

## 📂 兩種輸入法的格式對照

| 規則 | 自然輸入法 CSV | 華碩智慧輸入法 TXT |
|---|---|---|
| 副檔名 | `.csv` | `.txt` |
| 編碼 | UTF-8 **有 BOM** | UTF-8 **無 BOM** |
| 換行 | CRLF（\r\n） | CRLF（\r\n） |
| 詞與注音分隔 | 半形逗號 `,` | **兩個半形空格** |
| 注音音節之間 | 連字號 `-` | **無分隔，直接相連** |
| 一聲標記 | 不標 | 標 `ˉ` |
| 開頭標記 | `<!--Going12ProfileBackUpCsv-Begin-->,` | 無 |

### 同一個詞，兩種格式長這樣

```
詞彙：王小明

自然輸入法 CSV：
<!--Going12ProfileBackUpCsv-Begin-->,
王小明,ㄨㄤˊ-ㄒㄧㄠˇ-ㄇㄧㄥˊ

華碩智慧輸入法 TXT：
王小明  ㄨㄤˊㄒㄧㄠˇㄇㄧㄥˊ
```

## ⚠️ 注意事項

| 風險 | 說明與建議 |
|---|---|
| **破音字一定要手動檢查** | 工具預設選最常用讀音，但個人打字習慣可能不同。例如「血」可以是 `ㄒㄧㄝˇ`（口語）或 `ㄒㄩㄝˋ`（書面）、「中」可以是 `ㄓㄨㄥ`（中心）或 `ㄓㄨㄥˋ`（中風）、姓氏「許」應為 `ㄒㄩˇ`（pinyin-pro 預設為一聲，需手動改） |
| **冷僻字可能失敗** | 若 pinyin-pro 字典未收錄，會顯示「注音對照失敗」，需手動填注音 |
| **兩種輸入法的破音字可能要分別校正** | 同一個破音字，妳在兩種輸入法的詞庫可能會選不同讀音，記得**兩欄都檢查** |
| **首次載入需要網路** | 開啟頁面時會從 cdnjs 載入 pinyin-pro（約 1 MB），載入後即使離線也能繼續使用 |
| **本工具只處理備份／字典格式檔** | 自然輸入法輸出的 CSV 是給「匯入 CSV」用、華碩輸出的 TXT 是給「使用者字典匯入」用，不適用其他編輯場景 |
| **匯入後重複的詞會被忽略** | 兩種輸入法都會自動判斷重複，原本就有的詞不會被覆蓋（除非選「全部覆蓋」） |

## 📝 開發背景

此工具誕生於批次匯入 113 筆人名到自然輸入法詞庫時遇到的痛點：官方教學說「匯入時注音欄可留空、系統會自動配音」，但實測在備份格式的匯入路徑中**注音欄不可留空，必須手動填入連字號連接的注音碼**。

後續為了同時支援華碩智慧輸入法（格式規則完全不同），擴充為雙輸入法支援。

> 注音碼／鍵盤碼：輸入法用來記錄「打什麼注音會跳出這個詞」的對應碼，例如打 `ㄍㄢˇ-ㄇㄠˋ` 跳出「感冒」。
